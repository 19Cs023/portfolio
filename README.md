# Dhanush M's Portfolio

Modern personal portfolio built with React and Parcel.

## Overview

This portfolio showcases my background as a full stack developer, a short bio, a project gallery, and social/contact links. It is designed to be responsive and deployed to GitHub Pages.

## Live Site

- GitHub Pages: https://19cs023.github.io/portfolio

## Features

- Responsive layout for desktop and mobile
- Hero section with background image
- About section with skills and summary
- Project gallery with project cards
- Contact footer with social icons

## Tech Stack

- React
- Parcel
- HTML
- CSS

## Project Structure

- `src/App.jsx` - main app composition
- `src/Components/` - page sections and reusable UI pieces
- `src/styles.css` - global styles and responsive layout rules
- `src/index.html` - app shell used by Parcel

## Getting Started

### Prerequisites

- Node.js
- npm

### Install Dependencies

```bash
npm install
```

### Run Locally

```bash
npm start
```

### Build for Production

```bash
npm run build
```

### Build for GitHub Pages

```bash
npm run build-gh
```

### Deploy to GitHub Pages

```bash
npm run deploy
```

## Notes

- The GitHub Pages deployment uses the `gh-pages` branch.
- The production build uses relative asset paths so it works correctly on GitHub Pages.
